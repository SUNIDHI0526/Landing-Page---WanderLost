# 🌍 WanderLost – Travel Landing Page  

A sleek and responsive landing page for **WanderLost**, a luxury travel brand offering curated escapes, premium packages, and timeless memories. Built with **HTML5, CSS3, and JavaScript**, this project showcases destinations, packages, and customer reviews in a modern, user‑friendly design.  

---

## ✨ Features  
- 🏝️ **Hero Section** – Eye‑catching banner with tagline + CTA  
- 🗺️ **Popular Destinations** – Dubai, Bali, Goa, Paris, Kashmir with pricing  
- 📦 **Travel Packages** – Solo, Couple, Family, Friends, Luxury options  
- ⭐ **Traveller Reviews** – Authentic testimonials with ratings  
- 🏨 **Why Choose Us** – Premium hotels, personalized itineraries, safe travel  
- 📞 **Footer Section** – Contact info, office address, copyright  

---

## 🛠️ Tech Stack  
- **HTML5** – Semantic structure  
- **CSS3** – Styling & layout (Flexbox/Grid recommended)  
- **JavaScript** – Optional interactivity
